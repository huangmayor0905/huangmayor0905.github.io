今天來戒手機

線上課的第二天，總覺得每次在家裡都沒辦法專心做事，一直想看手機。

哈哈哈，結果只有上午成功，下午直接破功。

話說我這幾天沒更新不是我懶，我有在更新這篇用 Hexo 在 GitHub 架站吧，目前是更新完了，但後面的更換主題還沒準備。還有我居然看到有學長用他自己的部落格放空英單字，害得我也想放筆記在上面，歐對齁……我這個預設本來就是放筆記啊。
